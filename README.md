# Rock Paper Scissors Web Game

A simple interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## Features

- Player vs computer gameplay
- Random computer choice
- Live score tracking
- Reset score button
- Interactive UI

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

1. The player clicks Rock, Paper, or Scissors.
2. The computer randomly selects its move.
3. The game determines the winner.
4. Scores update dynamically on the page.
